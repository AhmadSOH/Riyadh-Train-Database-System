This is Group project for Fundamentals Of Database Systems (CSC380) Course in King Saud University,

the project was about making a database for a train station (we named it "riyadh station").

the steps of the project was:
1-introduction and planning for Data requirement, Entitles and attributes, Relationships and Business rules.
2-ER digram with 6 entities.
3-mapping.
4-apply it on SQL.
5-apply queries to the SQL code (SELECT, UPDATE, DELETE.. etc.).

we used MySQL in this project.

the project code are in the .sql file "Riyadh_Train",
all the results and queries are in the Report file.

the project was done in first semester of year 2020/2021 (Online Semester).
